# Family Tree Frontend

React + TypeScript + Vite frontend for the Zimbabwean Family Tree project.

## Prerequisites

- Node.js 20+
- Backend API running at `http://localhost:8000`

## Setup

```bash
npm install
cp .env.example .env
```

## Run

```bash
npm run dev
```

The app runs on `http://localhost:5173` by default.

## Environment

Set API base URL in `.env`:

```bash
VITE_API_BASE_URL=http://localhost:8000/api/v1
```

## Build

```bash
npm run build
```
